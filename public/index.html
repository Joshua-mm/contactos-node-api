Lista de rutas de peticiones

Usuarios

Crear un nuevo usuario: Peticion POST

https://api-contactos-zeik.herokuapp.com/usuario

Contenido del body - x-www-form-urlencoded. (Usar las mismas keys que se muestran en la imágen)

Ver información de un usuario: Petición GET

Para esta petición es necesario proporcionar un token mediante los header con el key 'token'

https://api-contactos-zeik.herokuapp.com/usuario

Actualizar información de un usuario: Petición PUT

https://api-contactos-zeik.herokuapp.com/usuario/:usuario_id

Contenido del body - x-www-form-urlencoded. (Usar las mismas keys que se muestran en la imágen)
Para esta petición es necesario proporcionar un token mediante los header con el key 'token'

Desactivar un usuario - Petición Delete

https://api-contactos-zeik.herokuapp.com/usuario/:usuario_id


Para esta petición es necesario proporcionar un token mediante los header con el key 'token' y ser un administrador de la aplicación
